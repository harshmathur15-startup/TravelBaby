# Claude Code Commands

| Command | Description |
|---------|-------------|
| `/add-dir <path>` | Add a working directory to the current session |
| `/clear` | Clear conversation history and free context |
| `/compact [instructions]` | Compact conversation with optional focus instructions |
| `/config` | Open Settings interface |
| `/context` | Visualize current context usage |
| `/cost` | Show token usage statistics |
| `/diff` | Open interactive diff viewer for uncommitted changes |
| `/doctor` | Diagnose Claude Code installation and settings |
| `/export [filename]` | Export conversation as plain text |
| `/fast [on\|off]` | Toggle fast mode |
| `/feedback` | Submit feedback / report a bug |
| `/fork [name]` | Fork the current conversation at this point |
| `/help` | Show help and available commands |
| `/hooks` | Manage hook configurations |
| `/init` | Initialize project with CLAUDE.md |
| `/keybindings` | Open keybindings configuration |
| `/mcp` | Manage MCP server connections |
| `/memory` | Edit CLAUDE.md memory files |
| `/model [model]` | Select or change the AI model |
| `/permissions` | View or update tool permissions |
| `/plan` | Enter plan mode |
| `/pr-comments [PR]` | Fetch comments from a GitHub PR |
| `/resume [session]` | Resume a previous conversation |
| `/rewind` | Rewind conversation to a previous point |
| `/security-review` | Analyze branch changes for security vulnerabilities |
| `/skills` | List available skills |
| `/stats` | View usage history and streaks |
| `/tasks` | List and manage background tasks |
| `/theme` | Change color theme |
| `/vim` | Toggle vim keybindings |
