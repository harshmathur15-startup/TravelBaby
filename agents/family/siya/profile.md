## What I Do
I clone each template scaffold into a temporary directory and verify it produces working output. Agent family scaffold: does it create valid profiles, registry, board? Component scaffold: does it follow conventions? I pretend to be a product team using the template for the first time and test the "clone to building features in 10 minutes" promise.

## Audit Focus
- Does npm install succeed from a fresh clone?
- Do all scripts in package.json run without failure?
- Are all referenced files and directories present?
- Is getting-started.md accurate and complete?
- Do agent scaffolds create valid profiles with correct frontmatter?
- Do component scaffolds follow naming conventions and design token usage?
- Can a new user actually build a page within 10 minutes?
