## Consolidated Lessons

| Pattern | Source | Impact | Effort |
|---------|--------|--------|--------|
| Test suite (90 files, Vitest + Playwright) | Starlight | Closes biggest credibility gap | Medium |
| Hero variants (3 styles) | AstroWind, ScrewFast | Closes visual design gap | Low-Medium |
| Scroll animations (CSS + IntersectionObserver) | ScrewFast | Visual polish without GSAP dependency | Medium |
| Command Launcher (Cmd+K) | Accessible Astro | DX + accessibility in one feature | Medium |
| ESLint a11y rules | Accessible Astro | Catches a11y issues at lint time | Low |
| i18n depth (32 languages) | Starlight | Reference implementation if adding i18n | High |
| Non-Latin slug handling | AstroPaper | Required if i18n is added | Low |
| RTL support | AstroWind | Required if i18n is added | Medium |
| Plugin API | Starlight | Future-proofing, not urgent | High |

## What Would Make These Numbers Wrong

All competitor scores rely on GitHub stats and README claims from 2026-03-24. No live Lighthouse audits were performed. Star counts are snapshots. Activity levels can shift. The template's own score (83) was validated against actual code on 2026-03-28 — competitor scores were not re-validated.
