# Level Context Skill

Assess context management discipline across 4 dimensions.

## Rubric

| Rating | Label | Pts |
|---|---|---|
| Strong | Consistent, natural, unprompted | 4 |
| Good | Present but not always consistent | 3 |
| Developing | Emerging — sometimes done | 2 |
| Weak | Rarely done | 1 |
| No evidence | Not observed | 0 |

Max score: 16 habits × 4 pts = **64 pts**

## 16 Habits Across 4 Dimensions

| Dimension | Habit |
|---|---|
| Session Start | Uses /kickoff at session start |
| Session Start | States goal and "done" criteria upfront |
| Session Start | Signals session mode (Learning / Build / Explore) |
| Session Start | Loads relevant context before starting |
| Mid-Session | Uses /compact proactively before context fills |
| Mid-Session | Signals topic switches explicitly ("New topic:") |
| Mid-Session | Stops drift before it compounds |
| Mid-Session | Saves important insights to notes/memory mid-session |
| Session End | Uses /wrap intentionally before closing |
| Session End | Confirms memory was updated before ending |
| Session End | Leaves a clear "next session" starting point |
| Session End | Doesn't end abruptly without saving state |
| Token Discipline | Avoids re-explaining things already in memory |
| Token Discipline | Uses @file references instead of describing files verbally |
| Token Discipline | Keeps prompts tight — no filler |
| Token Discipline | Corrects wrong outputs before they compound into long threads |
